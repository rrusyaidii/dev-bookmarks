# DevMark Chrome Extension

One-click save the current tab into your local DevMark app.

## Setup

1. Run DevMark locally: `npm run dev` (http://localhost:3000)
2. Open Chrome → `chrome://extensions`
3. Enable **Developer mode**
4. **Load unpacked** → select this `extension/` folder
5. Pin the extension, open any page, click **Save bookmark**

## Settings

Right-click the extension icon → Options, or open the Settings link in the popup.
Default API base: `http://localhost:3000`
